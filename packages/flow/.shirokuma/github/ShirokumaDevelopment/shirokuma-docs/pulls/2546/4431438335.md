### 修正内容

`pushIssueBody` の Status ガード（`validateTransition(allowRollback: false)`）を `setItemFields` 呼び出しより**前**に移動した。これにより:

- ロールバック遷移で Status ガードが拒否した場合、非 Status フィールドの書き込みは行わずに早期 return する
- Status と非 Status を同時に変更するケースで、ガード拒否時の部分更新を防止
